# ✈️ 高雄直飛峴港 - 便宜機票獵人 (Airfare Budget Hunter Skill)

## 📖 專案簡介
本專案旨在展示如何運用**流程最佳化**的思維，將原本耗時、繁瑣的「機票比價與行程規劃」過程，封裝成一套標準化、可重複執行的 AI 工作規範（Skill）。

專案以南部大學生背包客最熱門的「高雄小港 (KHH) ➡️ 越南峴港 (DAD)」航線為核心情境。透過設定明確的決策變數（預算上限、行李分配規則、在地出發優勢），引導 AI 自動排除不具效益的無效資訊，最終產出高性價比的決策報告與視覺化看板。

## 🎯 核心工作流程與決策邏輯
本 Skill 內建以下三大評估標準，確保產出結果符合實際需求：
* **在地主場優勢：** 強制鎖定小港機場直飛，省去前往北部搭機的隱性交通成本與時間耗損。
* **預算極大化配置：** 導入「行李彈性拆解法」（去程純手提、回程加購托運），並將簽證規費納入整體預算考量。
* **最佳時機捕捉：** 自動鎖定淡季促銷與週間離峰時段，避開無謂的票價溢價。

## 📂 專案架構 (Repository Structure)
* `SKILL.md`：本專案的核心靈魂，詳細定義了機票獵人的角色設定、判斷規則與輸出格式標準。
* `assets/`：存放使用者情境的輸入範本（如 `example_prompt.txt`），提供 AI 啟動流程的初始條件。
* `references/`：存放標準化的輸出格式範本（如 `itinerary_template.md`），確保 AI 產出的報告具備高度一致性。
* `demo/`：展示本 Skill 執行後的具體成果，包含：
  * `flight_report.md`（機票比價與搶票攻略報告）
  * `flight_dashboard.html`（互動式機票比價視覺看板）

## 💡 使用說明
1. 複製 `assets/example_prompt.txt` 中的需求描述。
2. 將需求與 `SKILL.md` 內的規則設定，一併輸入至您慣用的 AI 工具（如 ChatGPT 或 Claude）。
3. AI 將嚴格遵循本專案定義的工作流，為您生成專屬的機票決策報告！
